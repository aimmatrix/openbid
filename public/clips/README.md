# Demo clips

These three files back the Video Stage. They are currently **placeholder
animated gradients** (generated by `scripts/generate-placeholder-clips.sh`).

| File | Scene | Swap in real footage of… |
|---|---|---|
| `kitchen-morning.mp4` | `scene_kitchen_morning` | a warm kitchen at sunrise, someone making coffee, a plain mug in frame |
| `park-afternoon.mp4` | `scene_park_afternoon` | an outdoor park, picnic, **a child clearly visible** (this is what makes the safety block land) |
| `tier2-fallback.mp4` | Tier-2 fallback | any short branded ad clip (plays if live AI generation is slow/unavailable) |

## Swapping in real footage

Just drop a `.mp4` with the **same filename** here — no code changes. Filenames
must match `clip_url` in `mocks/scenes.ts`. Keep them short (5–10s), 16:9,
H.264, and silent/muted-friendly.

Sources: record on a phone, license stock (Pexels/Pixabay have free 16:9 clips),
or generate via the Tier-2 video provider.

## Regenerate the placeholders

```bash
brew install ffmpeg          # one-time
bash scripts/generate-placeholder-clips.sh
```
